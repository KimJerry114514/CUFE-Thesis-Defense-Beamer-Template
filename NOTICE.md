# Notice

This repository is an unofficial Beamer template for thesis defense presentations.

## Upstream Attribution

The style files in this project are historically derived from CUFE/THU Beamer template variants. Known upstream references include:

- THU Beamer Theme by Jiayi Weng: https://www.overleaf.com/latex/templates/thu-beamer-theme/vwnqmzndvwyb
- THU Beamer Theme GitHub page referenced by Overleaf: https://github.com/Trinkle23897/THU-Beamer-Theme
- Earlier local CUFE template notes referencing course/thesis LaTeX template adaptations.

The Overleaf page for THU Beamer Theme lists the license as Creative Commons Attribution 4.0 International (CC BY 4.0).

## Non-Official Status

This project is not an official template issued by Central University of Finance and Economics. It is provided for personal academic presentation use. Users should verify formatting requirements with their department, advisor, or teaching office before formal submission or defense.
